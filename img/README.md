# 🦅 A.D. Alcotán Canarias - Dashboard de Gestión 2026/2027

Este repositorio contiene la plataforma digital de captación y gestión estratégica para el relanzamiento de la **A.D. Alcotán Canarias**. El proyecto está diseñado bajo un modelo de gestión deportiva profesional, orientado a familias y centrado en la sede de **Cuevas Torres**.

## 🎯 Objetivo del Proyecto
Reconstruir la base deportiva del club mediante la creación de una estructura federada sólida para las categorías:
* **Prebenjamín**
* **Benjamín**
* **Alevín**

## 🛠️ Pilares de Gestión
El sitio web actúa como el centro operativo de captación y comunicación, destacando:
1. **Seguridad (LOPIVI):** Implementación de la figura del Delegado de Protección del Menor.
2. **Metodología:** Plan formativo propio con capacitación mensual para el staff técnico.
3. **Tecnología e IA:** Uso de herramientas digitales para optimizar la organización y el seguimiento deportivo de forma transparente.
4. **Sede Única:** Foco exclusivo en el Complejo Deportivo Cuevas Torres.

## 📁 Estructura del Repositorio
* `index.html`: Dashboard interactivo optimizado para dispositivos móviles (Responsive).
* `img/`: Directorio de recursos visuales.
    * `logo.png`: Logotipo corporativo del club (Negro y Amarillo).
* `README.md`: Documentación técnica y estratégica.

## 🚀 Implementación Técnica
La web utiliza:
* **Tailwind CSS:** Para un diseño "Premium Dark" ágil y profesional.
* **Chart.js:** Visualización de datos de viabilidad y estructura.
* **Google Forms Embed:** Sistema de captura de leads integrado directamente en la base de datos (Google Sheets).
* **IA Concienciada:** Diseño de interfaz y textos optimizados para el Buyer Persona (familias).

## 📅 Plan de Actuación (Abril - Junio)
1. **Abril:** Lanzamiento de la plataforma y pre-inscripciones digitales.
2. **Mayo:** Jornadas de puertas abiertas en Cuevas Torres y uso de tecnificación como embudo.
3. **Junio:** Torneo Alcotán Canarias y cierre de inscripciones federativas.
